## 操作方式變更

### 對使用 `useApiWithCredits` hook 的開發者

**舊方式（已廢棄）**：
```typescript
const { apiFetch } = useApiWithCredits();
```

**新方式**：
```typescript
const { refreshUser } = useAuth();
const { apiFetch } = useApiWithCredits({ onMutationSuccess: refreshUser });
```

> 如果元件不需要自動刷新點數，可省略 `onMutationSuccess` 參數。

### 對需要手動刷新使用者狀態的開發者

**舊方式（已廢棄）**：
```typescript
if ((window as any).refreshAuthUser) (window as any).refreshAuthUser();
```

**新方式**：
```typescript
import { useAuth } from '../context/AuthContext';
const { refreshUser } = useAuth();
// 需要刷新時直接呼叫
refreshUser();
```
