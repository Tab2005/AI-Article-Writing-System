# Anthropic Skills 功能總覽

本文檔整理了 Anthropic Skills 倉庫中所有技能的功能和描述。這些技能涵蓋了創意設計、開發技術、企業溝通和文檔處理等多個領域。

## 技能列表

### 演算法藝術 (algorithmic-art)
**名稱:** algorithmic-art  
**描述:** Generates algorithmic art using JavaScript and HTML5 Canvas  
**功能概述:** 此技能允許 Claude 使用 JavaScript 和 HTML5 Canvas 生成演算法藝術。包含生成器模板和查看器 HTML 文件，用於創建動態視覺藝術作品。

### 品牌指南 (brand-guidelines)
**名稱:** brand-guidelines  
**描述:** Ensures content follows brand guidelines and maintains consistency  
**功能概述:** 此技能幫助 Claude 遵循品牌指南，確保內容的一致性和品牌合規性。適用於企業溝通和內容創建。

### Canvas 設計 (canvas-design)
**名稱:** canvas-design  
**描述:** Creates visual designs using HTML5 Canvas with custom fonts  
**功能概述:** 使用 HTML5 Canvas 和自定義字體創建視覺設計。包含多種開源字體，用於專業設計工作。

### 文檔協作 (doc-coauthoring)
**名稱:** doc-coauthoring  
**描述:** Enables collaborative document editing and co-authoring workflows  
**功能概述:** 支持文檔協作編輯和共同創作工作流程，適用於團隊文檔處理。

### DOCX 文檔處理 (docx)
**名稱:** docx  
**描述:** Creates and edits Microsoft Word documents (.docx)  
**功能概述:** 創建和編輯 Microsoft Word 文檔。包含完整的 OOXML 架構支持和文檔處理腳本。

### 前端設計 (frontend-design)
**名稱:** frontend-design  
**描述:** Designs and implements modern web user interfaces  
**功能概述:** 設計和實現現代網頁用戶界面，專注於前端開發和設計。

### 內部溝通 (internal-comms)
**名稱:** internal-comms  
**描述:** Creates internal communications and announcements  
**功能概述:** 創建內部溝通和公告，包含示例和最佳實踐。

### MCP 建構器 (mcp-builder)
**名稱:** mcp-builder  
**描述:** Builds Model Context Protocol (MCP) servers and tools  
**功能概述:** 建構 MCP 服務器和工具，包含參考文檔和腳本。

### PDF 處理 (pdf)
**名稱:** pdf  
**描述:** Processes and analyzes PDF documents, including form extraction  
**功能概述:** 處理和分析 PDF 文檔，包含表單提取和參考文檔。

### PPTX 簡報處理 (pptx)
**名稱:** pptx  
**描述:** Creates and edits Microsoft PowerPoint presentations (.pptx)  
**功能概述:** 創建和編輯 PowerPoint 簡報，包含 HTML 到 PPTX 轉換和 OOXML 支持。

### 技能創建器 (skill-creator)
**名稱:** skill-creator  
**描述:** Creates new Claude skills and skill templates  
**功能概述:** 創建新的 Claude 技能和技能模板，包含參考文檔和腳本。

### Slack GIF 創建器 (slack-gif-creator)
**名稱:** slack-gif-creator  
**描述:** Creates animated GIFs for Slack communications  
**功能概述:** 為 Slack 溝通創建動畫 GIF，包含核心功能和需求文件。

### 主題工廠 (theme-factory)
**名稱:** theme-factory  
**描述:** Generates design themes and color palettes  
**功能概述:** 生成設計主題和顏色調色板，包含預定義主題。

### 網頁工件建構器 (web-artifacts-builder)
**名稱:** web-artifacts-builder  
**描述:** Builds web artifacts and components  
**功能概述:** 建構網頁工件和組件，包含腳本和工具。

### 網頁應用測試 (webapp-testing)
**名稱:** webapp-testing  
**描述:** Tests web applications and user interfaces  
**功能概述:** 測試網頁應用和用戶界面，包含示例和腳本。

### XLSX 試算表處理 (xlsx)
**名稱:** xlsx  
**描述:** Creates and edits Microsoft Excel spreadsheets (.xlsx)  
**功能概述:** 創建和編輯 Excel 試算表，包含重新計算功能。

## 使用說明

這些技能可以通過以下方式使用：

1. **Claude Code:** 安裝對應的插件
2. **Claude.ai:** 上傳或使用預建技能
3. **Claude API:** 通過 API 使用技能

每個技能都包含在各自的資料夾中，具有 `SKILL.md` 文件，其中包含詳細的使用說明和示例。

## 注意事項

這些技能僅供演示和教育目的。實際使用時可能與 Claude 的實現有所不同，請在使用前進行充分測試。
