# Seonize 圖片整合功能使用指南

本文件旨在說明 Seonize 系統中圖片整合功能的操作流程與技術特性，幫助使用者快速上手「圖文並茂」的內容創作。

---

## 1. 核心功能亮點

### 📸 AI 驅動的視覺建議 (Outline Stage)
在生成文章大綱時，AI 會同步根據內容產出「圖片主題建議」：
*   **視覺化提示**：在大綱編輯器中，每個章節旁會有 🖼️ 圖示，懸停即可查看 AI 規劃的配圖主題。
*   **關鍵字自動化**：AI 會為每個章節準備適合圖庫搜尋的**英文關鍵字**。

### 🤖 AI 自動化標籤與圖說 (Writing Stage)
在撰寫頁面插入圖片後，系統會自動處理 SEO 標籤：
*   **Alt Text 自動生成**：根據當前章節的具體內容，AI 會自動生成描述性的 Alt 標籤，大幅提升 Google 圖片搜尋的排名潛力。
*   **自動圖說 (Caption)**：系統會自動產出一段吸引讀者的圖說，並以斜體樣式呈現。

### ⚡ 圖片效能優化 (WebP)
*   所有手動上傳的圖片（JPG/PNG）在儲存時，系統會自動轉換為 **WebP** 格式。
*   此舉能減少約 50%-80% 的檔案大小，同時維持高品質，符合 Google Core Web Vitals 的載入速度要求。

### 🚀 WordPress 媒體庫全自動同步 (CMS Sync)
*   **雲端上傳**：發布至 WordPress 時，系統會自動偵測內容中的本地圖片並將其上傳至 WordPress 媒體庫。
*   **特色圖片 (Featured Image)**：文章中的**第一張圖片**會自動被設為 WordPress 的特色圖片，無需任何手動設定。

---

## 2. 操作流程

### 第一步：大綱生成與預覽
1.  進入「因果矩陣」或「關鍵字研究」生成大綱。
2.  在大綱編輯器中，確認每個章節是否有配圖建議圖標。

### 第二步：挑選並插入圖片
1.  在撰寫頁面點擊「插入圖片」。
2.  **圖庫搜尋**：搜尋框會自動帶入 AI 建議的關鍵字。點擊搜尋（需扣除 1 點）後挑選喜愛的圖片。
3.  **手動上傳**：也可以直接從電腦上傳圖片，系統會自動將其優化為 WebP。

### 第三步：發布至 WordPress
1.  完成文章後，點擊「發布」。
2.  選擇您的 WordPress 站點。
3.  發布成功後，前往 WordPress 後台，您會發現圖片已自動存在於媒體庫中，且網址已正確替換為雲端網址。

---

## 3. 環境變數要求 (部署用)

若要確保功能的完整執行，請在部署環境（如 Zeabur）設定以下變數：
*   `PEXELS_API_KEY`：啟用 Pexels 圖庫搜尋功能。

---

## 4. 點數扣除標準
*   **圖庫 API 檢索**：1 點 / 次。
*   **手動上傳與 AI 描述**：0 點。
*   **CMS 同步與 WebP 優化**：0 點。

---

> [!TIP]
> **SEO 建議**：
> 雖然系統會自動生成 Alt Text，但您依然可以在插入圖片前手動微調描述，以更精確地對齊您的核心目標關鍵字。
