# 劫之眼術 (Kalpa Eye) 功能說明文件

「劫之眼術」是 Seonize 系統中的核心內容策略推演模組。它透過「因果矩陣」的邏輯，將產業中的核心實體、使用者動作與常見痛點進行矩陣化掃描，自動生成具備高度搜尋意圖的內容計畫。

## 1. 核心組成要素

在進行因果推演前，系統需要定義以下三個維度：

*   **實體 (Entities)**：產業中的核心關鍵字、產品名、品牌名或平台（例如：幣安、MetaMask）。
*   **動作 (Actions)**：使用者在操作實體時執行的行為（例如：入金、提現、KYC 認證）。
*   **痛點 (Pain Points)**：使用者在執行動作時遇到的困難或負面體驗（例如：卡住、顯示錯誤代碼、等很久）。

## 2. 因果矩陣 (Matrix Generation)

系統會將上述三個維度進行交叉組合，生成「意圖節點」。
*   **推演邏輯**：每一個節點代表一個具體的搜尋場景。例如：`[幣安] + [入金] + [失敗]`。
*   **意圖標題**：系統會自動為每個節點生成符合 SEO 邏輯的文章標題。
*   **狀態追蹤**：分為「待編織」、「編織中」與「已編織」，方便管理內容產出進度。

## 3. 目標導航 URL (Money Page)

**「目標導航 URL」**（又稱 Money Page）是整個推演計畫的戰略核心。

### 作用與重要性
*   **最終轉化目標**：這是您希望使用者最後抵達的頁面（如：產品購買頁、服務註冊頁）。
*   **內容引導重心**：在執行「編織」功能時，AI 會確保生成的文章內容具備自然的引導邏輯，將流量導向此 URL。
*   **錨點優化**：系統會根據矩陣中的實體與動詞，自動優化指向此 Money Page 的內連（Internal Link）權重。

## 4. 內容編織 (Content Weaving)

「編織」功能是將意圖節點轉化為實際內容的過程。
*   **文章預覽**：編織完成後，開發者可直接預覽文章內容。
*   **法寶 (Anchors)**：系統會標註該篇文章所使用的法寶（關鍵字佈局策略），確保 SEO 效果最大化。

## 5. 因果查詢 (Saved Projects)

所有生成的矩陣都會儲存在系統中，方便日後隨時調研。
*   **專案管理**：可根據專案名稱、產業與儲存時間進行查詢。
*   **快速恢復**：點擊「查看詳情」後，系統會自動恢復該專案的推演狀態，讓您可以繼續進行內容編織或調整設定。
