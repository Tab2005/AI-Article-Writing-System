# Seonize 系統點數扣減與權限對照表

本文件記錄了 Seonize 系統內各項功能的點數消耗成本、會員等級權限以及相關折扣邏輯。

## 1. 點數消耗清單 (Credit Costs)

| 功能分類 | 操作項目 | 點數消耗 / 次 | 備註 |
| :--- | :--- | :--- | :--- |
| **關鍵字研究** | SERP 查詢 (無快取) | 2 | 深度反向工程搜尋結果 |
| | DataForSEO 數據獲取 | 3 | 獲取關鍵字成交意圖、搜尋量等 |
| | AI 搜尋意圖解析 | 2 | 分析使用者隱藏意圖 |
| **大綱與分析** | 建立 SEO 文章大綱 | 5 | 生成結構化大綱與邏輯鏈 |
| | 競品分析報告 | 3 | 分析對手 SERP 分佈 |
| | 內容缺口 (Content Gap) 分析 | 3 | 找出未覆蓋的競爭關鍵字 |
| **文章撰寫** | 生成單個段落 (Section) | 5 | 針對特定標題生成內容 |
| | **生成完整文章** | 20 | 包含 H1, Meta 與全文段落 |
| | 文章優化改寫 | 5 | 調整語氣或針對特定關鍵字優化 |
| | 全文品質審計 (Audit) | 3 | 檢查 SEO 達成率與可讀性 |
| **劫之眼 (Kalpa)** | 天道解析 (要素聯想) | 3 | AI 聯想實體、動作、痛點 |
| | 神諭編織：單節點成稿 | 8 | 根據矩陣節點生成完整解答 |
| | **神諭編織：批量成稿** | 8 / 節點 | 根據節點數量計算階梯折扣 |
| **CMS 發布** | AI 輔助排程優化 | 2 | 智慧判定最佳發布時機 |

## 2. 會員等級權限表 (Feature Access)

| 會員等級 | 代號 | 說明 | 限制功能 |
| :--- | :--- | :--- | :--- |
| **暫時試用 (Trial)** | 1 | 基礎功能體驗 | 不可使用：DataForSEO、完整文章生成、劫之眼批量成稿、CMS 集成 |
| **一般會員 (Basic)** | 2 | 標準商業應用 | 不可使用：劫之眼批量成稿 (Batch Weaving) |
| **深度會員 (Pro)** | 3 | 高階全自動化工具 | 無限制。享有批量成稿階梯折扣。 |
| **超級管理 (Admin)** | - | 系統最高權限 | 永遠不扣點。 |

## 3. 批量操作折扣規則 (僅限深度會員)

針對「劫之眼：批量成稿」功能，系統提供階梯式優惠：

| 批量節點數量 | 折扣率 | 實際成本計算 (示例) |
| :--- | :--- | :--- |
| 1 節點 | 100% (無折扣) | 8 點 |
| 2-5 節點 | 85% | 13.6 點 (2 節) ~ 34 點 (5 節) |
| 6-19 節點 | 80% | 38.4 點 (6 節) ~ 121.6 點 (19 節) |
| 20 節點以上 | 70% | 每節點僅需 5.6 點 (例如 50 節 = 280 點) |

> [!NOTE]
> 扣點邏輯採「先扣點、後執行」。若 AI 執行失敗或中途取消，系統將會呼叫退款 (Refund) 機制退還該次操作點數。
