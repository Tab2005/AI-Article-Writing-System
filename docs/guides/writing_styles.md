# Seonize 寫作風格與 AI 人格指南

本文件旨在說明 Seonize 系統中內置的「寫作風格」應用場景與背後邏輯。所有風格名稱均已規範化為三字格式。

## 🔹 正式寫作風格列表 (Official Styles)

| 風格名稱 | 風格特點 | 適用場景與理由 |
| :--- | :--- | :--- |
| **專業風** | 權威、中立、條理清晰。強調事實與數據，減少情感渲染。 | **常青指南、白皮書、專業領域指南**。適合建立品牌專業形象與信賴感，增強 E-E-A-T 信號。 |
| **評論風** | 評分、主觀分析、利弊對比。語氣較為直接且具備鑑定感。 | **產品評測、軟體對比、服務評價**。適合用於輔助使用者進行購買決策。 |
| **新聞風** | 簡潔、具時效性、客觀敘述。段落較短，聚焦於「發生了什麼」。 | **產業動態、政策更新、活動快訊**。適合發布最新的趨勢分析與即時消息。 |
| **對話風** | 親切、第一人稱、使用情緒詞。就像在跟朋友聊天一樣。 | **個人部落格、經驗分享、QA 問答**。能有效降低讀者的閱讀門檻，提升點擊率。 |
| **技術風** | 精確、邏輯嚴密。會大量使用專有名詞、代碼塊或規格表。 | **開發者文件、規格說明、專業術語解析**。針對特定專業領域的受眾，提供最高資訊密度。 |
| **開箱風** | 強調第一手體驗，包含優缺點分析、實測數據與真實經驗描述。 | **3C 產品實測、食品開箱、旅遊走訪**。高 E-E-A-T (經驗感)，極易獲得搜尋引擎青睞。 |
| **懶人包** | 極致結構化，開頭即結論。採用大量符號與短句，適合快速滑動。 | **政策重點、活動摘要、快速教學**。符合行動裝置閱讀習慣，易被抓取為精選摘要。 |
| **故事風** | 以問題場景勾起懸念，強調讀者感受（起承轉合），再帶入解決方案。 | **品牌敘事、心靈成長、B2C 行銷**。適合建立情感連結，提升品牌記憶點。 |

## 🛠️ 風格調整與維護建議
您可以透過「指令倉庫 (Prompt Warehouse)」中的 **AI 寫作人格 (Persona)** 設定來微調這些風格：

### 1. 確保自動匹配
由於風格名稱已更新為三字（如「專業風」），請確保您在「指令倉庫」中的標題或備註也包含這些關鍵字，以便 AI 寫作引擎能精確匹配到對應的人格。

### 2. 優化建議
*   **如果您覺得風格不夠鮮明**：可以在對應的人格模板中加入更具體的角色設定。
*   **混合使用**：建議針對不同意圖的關鍵字嘗試不同風格。例如，「商業型」意圖適合搭配「開箱風」或「評論風」。
